# Aflow Landing Page

Dependency-free bilingual static landing page for Aflow.

- `index.html`: document entry and metadata defaults.
- `product.html`: product family page for Aflow Agent and Specflow.
- `download.html`: dedicated download page.
- `styles.css`: visual tokens, layout, previews, responsive behavior and accessibility styles.
- `main.js`: shared bilingual content, rendering and locale/menu behavior.

Download links now route to `download.html`. Binary downloads open GitHub Releases, and command-line install snippets mirror the project README.
